> **The challenge:** Enterprise SOC tools assume x86_64. My machine is Apple Silicon (ARM64).
> **The outcome:** A fully documented set of workarounds published for the community.

```
MacBook Pro (Apple Silicon M-series)
└── VirtualBox
    └── Kali Linux ARM64
        ├── 🔍 SIEM          → Splunk (Docker/containerised) + ELK Stack 8.x
```
